Website penampil data hasil pengukuran sensor
EWSLongsor

EWSLongsor merupakan website penampil hasil pengukuran sensor dari web Thingspeak. Data hasil pengukuran sensor diolah oleh Arduino Uno, kemudian data dikirim menggunakan modul NodeMCU yang kemudian ditampilkan pada web Thinkspeak.
